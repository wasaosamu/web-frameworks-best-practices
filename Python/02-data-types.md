# 第2章：データ型

**この章の目標**  
Python でよく使う**データ型**を学びます。  
文字列、数値、リスト、辞書の扱い方を、初心者向けに説明します。

---

## データ型とは？

**データ型**とは、**データの種類**のことです。

- 「文字の並び」なのか、「数値」なのか、「複数のデータのまとまり」なのか  
- 型によって、できる操作が違う  

Python では変数に型を書かなくても、代入した値から自動で決まります。

---

## 文字列（str）

**文字列**は、**文字の並び**を表す型です。`"` や `'` で囲みます。

```python
# ダブルクォートでもシングルクォートでもよい
name = "山田"
message = 'こんにちは'

# 連結は + で
full_message = message + "、" + name + "さん"
print(full_message)  # こんにちは、山田さん

# 長さは len() で取得
length = len(name)
print(length)  # 2
```

---

## 数値（int, float）

**整数**は **int**、**小数**は **float** です。

```python
# 整数
age = 25
count = 10

# 小数
price = 99.9
pi = 3.14159

# 計算
sum = 10 + 5      # 15
diff = 10 - 5     # 5
product = 10 * 5  # 50
quotient = 10 / 5 # 2.0（割り算は常に float になる）
```

---

## リスト（list）

**リスト**は、**複数の値を並べたもの**です。`[ ]` で囲みます。

```python
# 数値のリスト
scores = [80, 90, 70]

# 文字列のリスト
names = ["山田", "佐藤", "鈴木"]

# 最初の要素は 0 番目
first = names[0]   # "山田"
second = names[1]  # "佐藤"

# リストの長さ
length = len(names)  # 3

# 要素の追加
names.append("田中")
# names は ["山田", "佐藤", "鈴木", "田中"] になる
```

---

## 辞書（dict）

**辞書**は、**キーと値のペア**を格納する型です。`{ }` で囲みます。

```python
# キー: 値 の形式
user = {
    "name": "山田",
    "age": 25,
    "email": "yamada@example.com"
}

# 値を取り出す
name = user["name"]   # "山田"
age = user["age"]     # 25

# 値を変更する
user["age"] = 26

# 新しいキーと値を追加する
user["phone"] = "090-1234-5678"
```

---

## 真偽値（bool）

**真偽値**は、**True** または **False** のどちらかです。

```python
is_active = True
is_empty = False

# 比較の結果も真偽値になる
age = 20
is_adult = age >= 20  # True
```

---

## 例題：データ型を組み合わせる

次の例は、リストと辞書を組み合わせて「ユーザー一覧」を表しています。  
各行にコメントを付けています。

```python
# ユーザーの辞書を3つ用意
user1 = {"name": "山田", "age": 25}
user2 = {"name": "佐藤", "age": 30}
user3 = {"name": "鈴木", "age": 22}

# リストにまとめる
users = [user1, user2, user3]

# 1人ずつ名前を出力
for user in users:
    print(user["name"], "さんは", user["age"], "歳です")
```

---

## まとめ

- **文字列**（str）は `"` や `'` で囲む。`+` で連結できる。
- **数値**は int（整数）と float（小数）。
- **リスト**（list）は `[ ]` で、複数の値を並べる。`[0]` で先頭にアクセス。
- **辞書**（dict）は `{ }` で、キーと値のペアを格納する。`["key"]` で値を取得。
- **真偽値**（bool）は True と False。
