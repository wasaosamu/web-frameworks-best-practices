**この章の目標**  
Python でよく使う**データ型**を学びます。  
文字列、数値、リスト、辞書の扱い方を、初心者向けに説明します。

---

## データ型とは？

**データ型**とは、**データの種類**のことです。

- 「文字の並び」なのか、「数値」なのか、「複数のデータのまとまり」なのか  
- 型によって、できる操作が違う  

Python では変数に型を書かなくても、代入した値から自動で決まります。

---

## 文字列（str）

**文字列**は、**文字の並び**を表す型です。`"` や `'` で囲みます。

```python
# ダブルクォートでもシングルクォートでもよい
name = "山田"             # ダブルクォートで文字列を定義
message = 'こんにちは'    # シングルクォートでも同様に定義可能

# 連結は + で
full_message = message + "、" + name + "さん"  # 文字列同士を + で連結
print(full_message)      # 連結結果を出力
# こんにちは、山田さん

# 長さは len() で取得
length = len(name)       # 文字列の文字数を取得
print(length)            # 2 を出力
```

---

## 数値（int, float）

**整数**は **int**、**小数**は **float** です。

```python
# 整数（int 型）
age = 25          # 整数リテラル
count = 10        # 整数リテラル

# 小数（float 型）
price = 99.9      # 小数点を含む数値
pi = 3.14159      # 円周率など

# 計算
sum = 10 + 5      # 加算 → 15
diff = 10 - 5     # 減算 → 5
product = 10 * 5  # 乗算 → 50
quotient = 10 / 5 # 除算（結果は常に float）→ 2.0
```

---

## リスト（list）

**リスト**は、**複数の値を並べたもの**です。`[ ]` で囲みます。

```python
# 数値のリスト
scores = [80, 90, 70]   # 角括弧 [] でリストを定義

# 文字列のリスト
names = ["山田", "佐藤", "鈴木"]  # 異なる型も混在可能

# 最初の要素は 0 番目（0始まりのインデックス）
first = names[0]   # インデックス 0 で先頭要素を取得 → "山田"
second = names[1]  # インデックス 1 で2番目の要素を取得 → "佐藤"

# リストの長さ
length = len(names)  # 要素数を取得 → 3

# 要素の追加
names.append("田中")  # 末尾に要素を追加
# names は ["山田", "佐藤", "鈴木", "田中"] になる
```

---

## 辞書（dict）

**辞書**は、**キーと値のペア**を格納する型です。`{ }` で囲みます。

```python
# キー: 値 の形式で辞書を定義
user = {
    "name": "山田",       # キー "name" に文字列を格納
    "age": 25,            # キー "age" に整数を格納
    "email": "yamada@example.com"  # キー "email" に文字列を格納
}

# 値を取り出す
name = user["name"]   # キーを指定して値を取得 → "山田"
age = user["age"]     # キーを指定して値を取得 → 25

# 値を変更する
user["age"] = 26      # 既存のキーの値を上書き

# 新しいキーと値を追加する
user["phone"] = "090-1234-5678"  # 存在しないキーに代入すると新規追加
```

---

## 真偽値（bool）

**真偽値**は、**True** または **False** のどちらかです。

```python
is_active = True      # 真を表す bool 型
is_empty = False      # 偽を表す bool 型

# 比較の結果も真偽値になる
age = 20              # 変数に数値を代入
is_adult = age >= 20  # 比較演算の結果が True になる
```

---

## 例題：データ型を組み合わせる

次の例は、リストと辞書を組み合わせて「ユーザー一覧」を表しています。  
各行にコメントを付けています。

```python
# ユーザーの辞書を3つ用意
user1 = {"name": "山田", "age": 25}  # 1人目のユーザー情報
user2 = {"name": "佐藤", "age": 30}  # 2人目のユーザー情報
user3 = {"name": "鈴木", "age": 22}  # 3人目のユーザー情報

# リストにまとめる
users = [user1, user2, user3]  # 辞書のリストで複数ユーザーを管理

# 1人ずつ名前を出力
for user in users:    # users の各要素を user に順次代入
    print(user["name"], "さんは", user["age"], "歳です")  # 辞書からキーで値を取得して出力
```

---

## まとめ

- **文字列**（str）は `"` や `'` で囲む。`+` で連結できる。
- **数値**は int（整数）と float（小数）。
- **リスト**（list）は `[ ]` で、複数の値を並べる。`[0]` で先頭にアクセス。
- **辞書**（dict）は `{ }` で、キーと値のペアを格納する。`["key"]` で値を取得。
- **真偽値**（bool）は True と False。
