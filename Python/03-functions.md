# 第3章：関数

**この章の目標**  
**関数**とは何か、どう書いてどう使うかを学びます。  
引数と戻り値の考え方を、初心者向けに説明します。

---

## 関数とは？

**関数**とは、**処理をひとまとまりにしたもの**です。

- 何度も使う処理を、1つにまとめておける  
- 「名前を呼ぶ」だけで、その処理を実行できる  
- **引数**でデータを渡し、**戻り値**で結果を受け取れる  

---

## 関数の書き方

```python
def 関数名(引数1, 引数2):
    # 処理を書く
    return 戻り値
```

- **def** は「関数を定義する」という意味  
- **引数**は、関数に渡すデータ（省略可能）  
- **return** で、関数の結果を返す（省略可能）  

---

## 引数なし・戻り値なしの関数

```python
def greet():
    print("こんにちは")

# 呼び出す
greet()  # こんにちは と出力される
```

---

## 引数ありの関数

```python
def greet(name):
    print("こんにちは、", name, "さん")

greet("山田")  # こんにちは、 山田 さん
```

---

## 戻り値ありの関数

```python
def add(a, b):
    result = a + b
    return result

sum = add(3, 5)
print(sum)  # 8
```

---

## デフォルト引数

**デフォルト引数**を使うと、引数を省略したときの値を決められます。

```python
def greet(name, message="こんにちは"):
    print(message, "、", name, "さん")

greet("山田")           # こんにちは 、 山田 さん
greet("佐藤", "おはよう")  # おはよう 、 佐藤 さん
```

---

## 例題：複数の処理を関数にまとめる

次の例は、ユーザー情報を表示する関数と、年齢判定する関数を定義しています。  
各行にコメントを付けています。

```python
def show_user(name, age):
    # ユーザー情報を表示する関数
    print("名前:", name)
    print("年齢:", age)

def is_adult(age):
    # 20歳以上なら True を返す
    return age >= 20

# 使う
show_user("山田", 25)
# 名前: 山田
# 年齢: 25

result = is_adult(25)
print(result)  # True
```

---

## まとめ

- **関数**は `def 関数名(引数):` で定義する。
- **引数**でデータを渡し、**return** で結果を返す。
- **デフォルト引数**で、省略時の値を指定できる。
- よく使う処理は関数にまとめると、コードが読みやすくなる。
