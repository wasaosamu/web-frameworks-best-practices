# 第1章：Python とは・基本の書き方

**この章の目標**  
Python の**基本的な書き方**を学びます。  
変数の宣言、画面への出力、コメントの書き方などを、初心者向けに説明します。

---

## Python とは？

**Python** は、**読みやすさを重視したプログラミング言語**です。

- **文法がシンプル**で、英語に近い書き方  
- **インデント（字下げ）**でブロックを表す（`{ }` を使わない）  
- Web アプリ、データ分析、自動化など、幅広く使われる  

---

## 変数とは？

**変数**とは、**データを入れておく箱**のようなものです。

- 名前を付けて、値を入れておける  
- あとからその名前で、中の値を取り出せる  
- Python では、**型を宣言しなくてよい**（代入した値から自動で決まる）

---

## 変数の書き方

```python
# 変数名 = 値 で代入する
name = "山田太郎"      # 文字列を変数に代入
age = 25              # 整数を変数に代入
is_active = True      # 真偽値を変数に代入
```

- **name** には文字列 `"山田太郎"` が入る  
- **age** には数値 `25` が入る  
- **is_active** には真偽値 `True` が入る  

Python では `=` の右側の値から、変数の型が自動で決まります。

---

## 画面への出力（print）

**print** を使うと、変数や文字列を画面に表示できます。

```python
# 文字列をそのまま出力
print("こんにちは")   # 引数の文字列を標準出力に表示

# 変数の値を出力
name = "山田"         # 変数に文字列を代入
print(name)           # 変数の内容を出力

# 複数をカンマで並べると、スペース区切りで出力される
age = 25              # 年齢を変数に代入
print(name, "さんは", age, "歳です")  # 複数引数はスペース区切りで連結されて出力
# 出力: 山田 さんは 25 歳です
```

---

## コメントの書き方

**コメント**は、コードにメモを残すためのものです。**実行されません**。

```python
# この行はコメント。プログラムの実行には影響しない

name = "山田"  # 行の後半にもコメントを書ける（インラインコメント）

# 複数行のコメントは
# 行の先頭に # を付けて書く
```

- 1行コメントは **#** の後ろに書く  
- 複数行は、各行の先頭に **#** を付ける  

---

## インデント（字下げ）について

Python では、**インデント**で「ブロック」を表します。

```python
# if の下は、4スペース（または1タブ）で字下げする
if age >= 20:         # 条件：age が 20 以上なら
    print("成人です")  # 条件が真のとき実行（インデント内）
    print("お酒が飲めます")  # 同上
# 字下げを戻すと、if のブロックの外
print("処理終了")     # 条件に関係なく常に実行
```

- **コロン（:）** の次の行は、字下げする  
- 字下げを戻すと、そのブロックの外に出る  
- 一般的に **4スペース** が推奨される  

---

## 例題：はじめての Python プログラム

次の例は、変数を宣言し、条件分岐でメッセージを変えて出力するプログラムです。  
各行にコメントを付けています。

```python
# ファイル名の例: hello.py

# 変数に名前と年齢を代入
name = "山田太郎"     # ユーザー名を格納
age = 20              # 年齢を格納

# 画面に出力
print("こんにちは、", name, "さん")  # 挨拶メッセージを表示

# 年齢に応じてメッセージを変える
if age >= 20:         # 20歳以上か判定
    print("成人ですね")   # 真の場合のメッセージ
else:                 # 偽の場合
    print("まだ未成年ですね")  # 偽の場合のメッセージ
```

---

## まとめ

- **変数**は `変数名 = 値` で代入する。型の宣言は不要。
- **print()** で画面に出力できる。
- **#** でコメントを書ける。実行には影響しない。
- **インデント**でブロックを表す。コロン（:）の次の行は字下げする。
