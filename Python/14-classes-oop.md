# 第14章：クラスとオブジェクト指向

**この章の目標**  
**クラス**の基本を学びます。  
データと処理をまとめた「型」を自分で定義する方法を、初心者向けに説明します。

---

## クラスとは？

**クラス**は、**データと処理をひとまとまりにした設計図**のようなものです。

- 「ユーザー」や「商品」など、同じ形のものが複数あるときに使う  
- クラスから作った実体を**インスタンス**と呼ぶ  
- 多くのライブラリやフレームワークでクラスが使われている  

---

## クラスの定義

```python
class User:
    def __init__(self, name: str, age: int):
        # __init__ は「インスタンスを作るときに呼ばれる」特別なメソッド
        self.name = name  # self は「このインスタンス自身」を指す
        self.age = age

    def greet(self) -> str:
        return f"こんにちは、{self.name} さん"
```

- **self** は、そのインスタンス自身を指す（必ず第1引数に書く）  
- **__init__** はコンストラクタ。インスタンス作成時に実行される  

---

## インスタンスの作成と利用

```python
user = User("山田", 25)
print(user.name)    # 山田
print(user.greet()) # こんにちは、山田 さん
```

- **User("山田", 25)** でインスタンスを作る（このとき `__init__` が呼ばれる）  
- **user.name** で属性にアクセス、**user.greet()** でメソッドを呼ぶ  

---

## 継承

**継承**は、既存のクラスをベースに新しいクラスを作る仕組みです。

```python
class AdminUser(User):
    def __init__(self, name: str, age: int):
        super().__init__(name, age)  # 親クラスの __init__ を呼ぶ

    def greet(self) -> str:
        return f"管理者の {self.name} です"
```

- **AdminUser(User)** で User を継承  
- **super()** で親クラスのメソッドを呼び出せる  

---

## まとめ

- **クラス**はデータと処理をまとめた設計図。**インスタンス**が実体。
- **__init__** で初期化、**self** で自分自身を参照する。
- **継承**で既存クラスを拡張できる。
